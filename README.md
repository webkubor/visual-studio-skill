# 🎨 视觉设计工作室 (Visual Design Studio)

[![Gemini CLI](https://img.shields.io/badge/Gemini--CLI-Skill-blueviolet)](https://github.com/google-gemini/gemini-cli)

**视觉设计工作室 (Visual Design Studio)** 是一个高度集成的全链路视觉资产生产中心。它合并了原本分散的绘图、分镜、Logo 设计与 UI 还原能力，为 AI 提供统一的审美准则与执行流程。

## 🛠 资源构成

- `SKILL.md`: 综合调度中心。
- `references/`:
  - `image-gen.md`: 绘图与锁相内核。
  - `storyboard.md`: 剧本与分镜内核。
  - `brand-ip.md`: 品牌一致性内核。
  - `logo-standard.md`: 矢量 Logo 规范。
  - `ui-ux.md`: 前端设计还原内核。
- `scripts/`: `setup-mcp.sh` (自动安装依赖)。

## 🚀 安装指南

```bash
gemini skills install .
```

本技能安装时会**自动尝试安装** `nanobanana` MCP 依赖。

## 📖 核心专家

1. **AI 绘图专家**: 调度 nanobanana 进行高质产出。
2. **剧本分镜专家**: 优化视觉叙事逻辑。
3. **品牌 IP 专家**: 守护品牌视觉 SSOT。
4. **前端设计专家**: 像素级设计还原。
