# 🎨 全链路视觉设计专家 (Visual Design Studio)

[![Gemini CLI](https://img.shields.io/badge/Gemini--CLI-Skill-blueviolet)](https://github.com/google-gemini/gemini-cli)

**Visual Design Studio** 整合了 AI 绘图、Logo 设计、UI 像素级还原等全链路视觉能力，由顶级创意总监内核驱动。

## 🚀 安装指南
```bash
gemini skills install https://github.com/webkubor/visual-design-studio
```

## 🔄 更新与维护
- **检查更新**: `npm run check-update`
- **一键升级**: `npm run update`
