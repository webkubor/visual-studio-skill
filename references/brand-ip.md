# 🎨 MZ MESO 品牌 IP 设计技能 (Brand IP Design SOP)

## 0. 品牌背景 (Brand Identity)
- **品牌名称**: **MZ MESO**
- **核心产品**: **竹纤维猫砂** (Bamboo Fiber Cat Litter)
- **核心卖点**: 天然环保、强效除臭、竹制抑菌、脚感舒适。

## 1. 核心设计原则 (UCD Based)
- **辨识度 (Clarity)**: 轮廓必须极简，3米外能一眼认出。
- **人格化 (Persona)**: 每个 IP 必须有性格锚点（如：倔强、贪吃、佛系）。
- **延展性 (Scalability)**: 必须能轻松适配 2D 漫画、3D 渲染和实体周边。

## 2. 视觉锁相协议
- **色谱控制**: 严格锁定 2 个主色 + 1 个辅助色。
- **特征器官**: 锁定一个非人类特征（如：发光的尾巴、异色瞳、特殊的围巾）。

## 4. 包装设计适配 (Packaging Specifics)
- **色彩管理**: 采用 Pantone 色值，**MZ MESO 竹纤维猫砂**建议以“生机绿 (#2D5A27)”和“象牙白”为主色，体现天然竹子原料。
- **构图留白**: IP 形象必须具备“抠图友好性”，背景干净，方便在包装袋上叠放文字（如：除臭、吸水率）。
- **材质暗示**: 视觉上通过 IP 的蓬松度暗示猫砂的柔软脚感，通过竹叶元素暗示气味清新。

## 5. 极简设计系列 (Minimalist Design Series)

### 5.1 候选角色 (2026-02-24 专属版)
1. **[阿竹 - Bamboo]**:
   - **视觉**: 极简线条构成的竹笋，代表**MZ MESO**的天然竹纤维原材料。
   - **性格**: 纯净、坚韧，象征产品的高效抑菌性能。
2. **[Meso 猫 - Meso Cat]**:
   - **视觉**: 只有耳朵和胡须线条的猫咪头部剪影，体现“隐形陪伴”感。
   - **性格**: 挑剔、优雅，代表对猫砂品质有极致追求的猫主子。
3. **[纤纤 - Fiber]**:
   - **视觉**: 几条交织的弧线，模拟竹纤维微观结构，极具科技感。
   - **性格**: 细心、专业，代表产品的超强吸水和除臭技术。
